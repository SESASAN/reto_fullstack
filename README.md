# OBSIDIAN E-commerce

E-commerce SPA desarrollado con React, TypeScript y Firebase. Tema oscuro premium con Tailwind CSS.

## Caracteristicas

- Catalogo de productos desde FakeStore API
- Buscador en tiempo real con paginacion
- Carrito de compras persistente
- Checkout con formulario de envio
- Autenticacion con Firebase Auth
- Ordenes guardadas en Firestore
- Diseno responsive para movil
- Menu movil desplegable
- Productos destacados (pagina independiente)

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Zustand (estado global)
- Firebase Auth + Firestore
- FakeStore API

## Deploy

Build: `bun run build`  
Output: `dist/`

Listo para Cloudflare Pages o Firebase Hosting.
